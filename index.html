📊 Infografía: Taller 2.2 – SQL JOINs

🎯 Objetivo

Fortalecer la extracción de información multitabla usando INNER JOIN, LEFT JOIN y RIGHT JOIN en bases de datos relacionales.

📋 Descripción

  • 20 consultas SQL en Northwind y Pubs
  • 8 INNER JOIN
  • 1 LEFT JOIN
  • 1 RIGHT JOIN

💻 Ejemplos

🟢 INNER JOIN

SELECT c.CompanyName, o.OrderID
FROM Customers c
INNER JOIN Orders o 
ON c.CustomerID = o.CustomerID;

Clientes con pedidos

🟡 LEFT JOIN

SELECT a.au_lname, ta.title_id
FROM authors a
LEFT JOIN titleauthor ta 
ON a.au_id = ta.au_id
WHERE ta.title_id IS NULL;

Autores sin libros

🔴 RIGHT JOIN

SELECT p.ProductName, od.OrderID
FROM Products p
RIGHT JOIN [Order Details] od 
ON p.ProductID = od.ProductID
WHERE od.OrderID IS NULL;

Productos no vendidos

🧠 Conclusiones

  • Los JOIN permiten integrar datos de múltiples tablas.
  • INNER JOIN es el más utilizado.
  • LEFT JOIN identifica datos faltantes.
  • RIGHT JOIN detecta inconsistencias.
  • Son esenciales en bases de datos reales.
